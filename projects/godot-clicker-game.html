Godot Clicker Game

This was the first game I made completely independently within the Godot game engine. I chose to make a very simplistic clicker game as I figured that I would be able to make something playable within a day. The gameplay side of the game is not completely balanced and thought out as I was primarily focusing on the technical side of the game and learning how Godot works.

Since I already know how to use C#, I decided to try using that with Godot, although I have followed some tutorials using GDScript.

This project allowed me to learn and practice how to use certain elements of the engine such as setting up GUIs as well as instancing of scenes in Godot.
