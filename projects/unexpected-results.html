Unexpected Results (Jun-2020)

Unexpected results is a point and click cyclical puzzle game made for a 'One Room' college-hosted game jam. This was made as part of a team. I was responsible for programming and assembling the game within the game engine. This game jam also allowed our work to be seen by industry professionals who reviewed and provided feedback for our game.

In the game, you need to find the key in each level to escape the room. Later levels increase difficulty by adding extra steps to find the key, such as needing to find a file to cut open the chair, for example.
